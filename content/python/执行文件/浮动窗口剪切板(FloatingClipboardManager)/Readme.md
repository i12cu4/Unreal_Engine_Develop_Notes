# Floating Clipboard Manager

## 描述

一个始终置顶的浮动剪贴板管理工具，可以快速保存和复制常用文本片段。

A always-on-top floating clipboard manager for quickly saving and copying frequently used text snippets.

## 功能特性

- 🪟 **浮动置顶窗口** - 始终显示在最前面
- 📋 **快速复制** - 一键复制保存的文本
- 💾 **持久化存储** - 自动保存到本地文件
- 🗑️ **便捷管理** - 轻松添加和删除条目
- 🔍 **悬停提示** - 显示完整文本内容
- 📱 **滚动支持** - 支持大量条目的滚动浏览

## 安装要求

- Python 3.6+
- 需要安装 `pyperclip` 库：

```bash
pip install pyperclip
```

## 使用方法

### 启动程序

```bash
python main.py
```

### 基本操作

1. **添加文本**：
   - 在输入框中输入文本
   - 点击"载入"按钮或按回车键

2. **复制文本**：
   - 点击对应的文本按钮
   - 文本会自动复制到剪贴板

3. **删除条目**：
   - 点击条目右侧的"删除"按钮

4. **查看完整文本**：
   - 将鼠标悬停在按钮上查看完整内容

### 数据存储

- 所有文本自动保存到 `saved_strings.json` 文件
- 程序启动时自动加载之前保存的文本
- 关闭程序时自动保存当前所有文本

## 技术细节

- 使用 Tkinter 创建图形界面
- 使用 JSON 文件进行数据持久化
- 支持鼠标滚轮滚动
- 自动处理长文本显示

## 注意事项

- 确保有写入当前目录的权限
- 文本过长时会被截断显示（完整内容可通过悬停查看）
- 程序关闭时会自动保存数据

## 故障排除

**复制功能失效**：
- 检查是否安装了 `pyperclip` 库
- 确认系统剪贴板权限

**无法保存数据**：
- 检查当前目录写入权限
- 确认磁盘空间充足

**界面显示异常**：
- 尝试调整窗口大小
- 重启应用程序
